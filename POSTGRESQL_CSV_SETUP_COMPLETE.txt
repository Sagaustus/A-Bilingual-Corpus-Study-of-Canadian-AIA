  ✅ No data redundancy (department info stored once in projects)

EXAMPLE QUERIES (AFTER IMPORT)
================================================================================

Find EN/FR governance divergences:
  SELECT p_en.project_title, g_en.oversight_mechanism, g_fr.oversight_mechanism
  FROM projects p_en
  JOIN projects p_fr ON p_en.project_title = p_fr.project_title
  LEFT JOIN governance g_en ON p_en.project_id = g_en.project_id
  LEFT JOIN governance g_fr ON p_fr.project_id = g_fr.project_id
  WHERE p_en.language = 'en' AND p_fr.language = 'fr';

Count rows by table:
  SELECT 'projects', COUNT(*) FROM projects
  UNION ALL SELECT 'governance', COUNT(*) FROM governance;

Find systems with no oversight:
  SELECT p.project_title FROM projects p
  LEFT JOIN governance g ON p.project_id = g.project_id
  WHERE g.oversight_mechanism IS NULL;
